# debate

Structured multi-round debate between AI tools with proposer/challenger roles and verdict

## Installation

```bash
# Claude Code
claude mcp add-plugin agent-sh/debate

# Or install from marketplace
agentsys install debate
```

## Usage

```
/debate
```
